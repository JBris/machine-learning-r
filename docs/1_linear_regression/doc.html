Linear Regression

Iris Dataset

We will demonstrate how to fit a linear regression model to the iris dataset.

First we will load the data.

library(knitr)
library(tidyverse)

data(iris)

iris = iris |>
  rename_with(~ tolower(gsub(".", "_", .x, fixed = TRUE)))

Next, we will visualise the data.

Regression
